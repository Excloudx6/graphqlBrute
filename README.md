### RUN

> Clone
```
git clone https://github.com/phor3nsic/graphqlBrute.git
```
> Install requirements

```
cd graphqlBrute && pip3 install -r requirements.txt
```

> Run
```
python3 bruteql.py -u http://target/graphql
```
